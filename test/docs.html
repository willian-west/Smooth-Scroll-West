Getting Started

No matter if you are a beginner or an advanced user, starting with SSW is easy..

Ease As



                            <body>


                                <main class="smooth-scroll-container">

                                    <div class="smooth-scroll-content">

                                    

                                        <!-- ALL YOUR CONTENT HERE -->

                                    

                                    </div>

                                </main>


                                <!-- position: fixed elements can go outside -->


                            </body>

Welcome to the documentation for SSW. This library provides advanced smooth scrolling, parallax effects, and moving the page to a certain point. Below, you'll find detailed information on how to set up and use each feature.

Important information:

For Mobile, SSW is disabled and the only function that still works is Parallax.

First Step

Let's go

To get started with Smooth Scroll West, make sure your HTML structure includes the required elements listed above.

Include the following JS file in your HTML:

Code 
<script src="path/to/smooth-scroll-west.min.js"></script>

Include the following CSS file in your HTML:

Code 
<link href="path/to/smooth-scroll-west.min.css" rel="stylesheet"></link>

Scroll Options

Customize the scrolling behavior with the following options:

  • scrollAmountMove: Adjust the scroll speed multiplier (default: 1.2).
  • scrollMoveTime: Set the scroll animation duration in milliseconds (default: 1100).
  • activeScrollPage: Enable or disable scrolling (default: true).

Example of changing options:

Code 

                                    __Scroll.scrollAmountMove = 1.5;

                                    __Scroll.scrollMoveTime = 1000;

Parallax Effects

Add parallax effects to your elements using the following classes:

  • js-parallax: Vertical parallax (requires data-start and data-end attributes). Value in pixels.
  • js-parallax-h: Horizontal parallax (requires data-offset and optional data-direction attributes). Value in pixels.
  • js-parallax-z: Zoom parallax (requires data-zoom attribute). Value from 1 to 100.

Example for vertical parallax:

Code 
<div class="js-parallax" data-start="-100" data-end="100"></div>

Example for horizontal parallax:

Code 
<div class="js-parallax-h" data-offset="200"  data-direction="right"></div>

Example for zoom parallax:

Code 
<div class="js-parallax-z" data-zoom="20"></div>

Scroll to Hash

The library supports smooth scrolling to elements based on the URL hash. Ensure your target elements use the scroll-id attribute instead of id. Smooth scrolling is disabled on Mobile, in this case for Mobile use the id attribute.

Example:

URL: https//domain.com/#contact

Code DESKTOP 
<div scroll-id="contact">Contact Section</div>
Code MOBILE 
<div id="contact">Contact Section</div>

Scroll to Section

The library supports smooth scrolling for elements based on id, for links on the same page.
Add the smooth-scroll-link class to the desired link. Make sure your target element has the corresponding id attribute.

Example:

Code 

                                    <a href="#contact" class="smooth-scroll-link">Contact</a>
Code 

                                    <section id="contact"></section>

The default time is 2000ms, but it can be changed using the data-smooth-time-link attribute. Value in milliseconds.

Code 

                                    <a href="#contact" class="smooth-scroll-link" data-smooth-time-link="5000">Contact</a>

Use Google Maps

Example

The behavior of the mouse wheel with the Google iframe interferes with the page scroll behavior, so to minimize the effects use the classes below:

Code 


                            <div class="map-container">

                                <div class="map-iframe">

                                    <!-- YOUR IFRAME HERE -->

                                </div>

                            </div>
