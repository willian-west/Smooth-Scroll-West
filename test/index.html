Smooth Scroll West

Smooth Scroll allows you to create a very smooth scrolling for your web page.

And it offers some other interesting features, such as:

  • Vertical parallax
  • Horizontal parallax
  • Anchoring, moving the page to a certain point, indicated by a Class or ID.
  • Reading the URL hash [ domain.com/#contact ] to move to the indicated element ID.
  • Adds or removes the 'is-hide' class for the Header tag, to show or hide the site menu, if it is in the Header tag during scrolling.

Important information:

Smooth Scroll does not work on mobile devices, as they already have smooth scrolling by default.

Vertical Parallax

The element will move in sync with the page scroll, within the maximum and minimum pixel limits. To activate it, simply add the "js-parallax" class to the desired element along with the "data-start" and "data-end" attributes, which indicate the starting and ending points of the movement. The movement will only occur when the element is within the visible area of ​​the page.

EXAMPLE 01

Title Parallax

Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatibus expedita delectus in labore, eos eius optio, at voluptas quidem dolor quam fugit iusto laboriosam eum reprehenderit laudantium porro repellendus praesentium!

Code 
 class="js-parallax" data-start="-50" data-end="50"
