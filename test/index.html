Smooth Scroll West

Smooth Scroll West allows you to create a very smooth scrolling for your web page with Parallax features.

Download GitHub

Version 1.0.0

It has some other interesting features, such as:

  • Vertical parallax
  • Horizontal parallax
  • Anchoring, moving the page to a certain point, indicated by ID.
  • Reading the URL hash [ domain.com/#contact ] to move to the indicated element ID.
